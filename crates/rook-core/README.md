# rook-core

Core library for Rook - traits, types, and memory implementation.

Rook is a long-term memory system that applies cognitive science principles to help AI assistants remember, forget, and retrieve information like humans do.

## Features

- FSRS-6 memory dynamics with spaced repetition
- Prediction error gating for smart ingestion
- Graph memory with spreading activation
- Hybrid retrieval modes
- Classification and consolidation

See the [main repository](https://github.com/BangRocket/rook) for full documentation.
